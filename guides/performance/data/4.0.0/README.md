# SAPL 4.0.0 performance data (archived)

These are the benchmark data files that backed the performance guide charts for
SAPL 4.0.0. Archived 2026-06-28 before regenerating the live `../*.json` files
for 4.1.0. Kept for reference only; the charts read the files in the parent
`data/` directory, not this folder.

Source files dated 2026-04-08. Measurement methodology (hardware, runtime,
profiles) is described in the performance guide. Note that these numbers were
produced on an earlier toolchain than the 4.1.0 set, so they are not a clean
version-to-version comparison.
